## Overview
This project is the back-end of a web-based platform designed to manage library resources and higher education course information efficiently. Built using **ASP.NET MVC**, **C#**, and **MSSQL Server**, it handles authentication, data storage, and CRUD operations while providing secure and reliable functionality for both users and admins.

## Features
- **User Authentication:** Secure sign-in and role-based access for admins and guests.
- **CRUD Operations:** Add, edit, delete, and manage books, CDs, and course information.
- **Reservation & Return Management:** Track and update item reservations and returns.
- **Feedback & Messaging:** Admins can view, respond, and manage messages from users.
- **Database Integration:** MSSQL Server database with optimized tables for efficient data storage and retrieval.
- **Validation & Security:** Server-side validation to prevent invalid input and ensure data integrity.

## Technologies
- **Back-End Framework:** ASP.NET MVC, C#
- **Database:** MSSQL Server
- **Data Access:** ADO.NET / Entity Framework for database operations
- **Security:** Role-based access, input validation, and secure query handling
